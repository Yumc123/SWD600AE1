    Web Developer - Design

    Posted 16 march by Randstad Employment Bureau

    Summary

    This is an exciting opportunity to be part of a close-knit international team who design, produce and publish global web content for a prestigious international organization. This role includes hands-on web publishing using enterprise grade tools.

    Job Responsibilities

    As part of the web project team, the web production assistant's key responsibilities will be:

  • • Monitoring web content and reporting weekly on quality utilising Digital Quality Management dashboards;
  • • Ideal candidate will be confident with recording actions from face to face meetings, planning successful meetings and responding to stakeholder requirements;
  • • Arranging for updates to be made to address issues identified during monitoring;
  • • Perform review and peer review of the content;
  • • Work with the Project Manager to execute the tasks related to specific projects;
  • • Hands on web publishing using WCS;

    • Required Experience and Skills

  • • Demonstrate a good understanding of authoring HTML code and de-bugging code errors
  • • Basic Adobe Photoshop and Dreamweaver proficiency
  • • Web Content Management knowledge and skills, preferably Oracle Web Center Sites
  • • Awareness of W3C standards.
  • • Understanding of cross-browser and cross-device testing criteria.
  • • Willingness and ability to adhere to corporate coding standards and branding guidelines.
